# Grafo

<p>
    * Um grafo G = (V,A) é constituido de um conjunto V de vertices e um conjunto A de arestas conectadas pares de vertices.<br/>
    * Quanto à orientação das arestas, este pode ser direcionado ou não direcionado.<br/>
